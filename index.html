Projects

Expat Journal - Capture

A proof-of-concept project for an Instagram competitor called Capture. With the help of one other partner and a UX designer, built the landing page and the "about us" section of the marketing page using HTML and CSS from scratch.

Med Cabinet - Dr. Mary Jane

Website that allows medical marijuana users to sort and filter through a large list of strains, and use a form that allows them to generate a list of strains based on their medical condition, preferred strains and preferred effects. Using React, I built and styled the sign-in, strain list, cabinet, and recommendations list pages. I also designed all the SVG icons on the site, including the logo. The algorithm that generates the recommendations is thanks to a talented team of data science developers.

Todo App 2

A simple reducer-based React app for managing todos. I intend to update this to sort todos by tag name and create something that has a worthwhile place on my portfolio.
